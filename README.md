# Jogo do Número Secreto 🎮🤖

## 🎯 Visão Geral
O **Jogo do Número Secreto** é uma aplicação web interativa onde o jogador deve adivinhar um número gerado aleatoriamente pelo sistema (de 1 a 10). O jogo fornece dicas se o número oculto é maior ou menor que o palpite atual, contabiliza as tentativas e utiliza síntese de voz para ler as instruções e resultados para o usuário.

## ✨ Funcionalidades
- **Lógica de Sorteio Inteligente:** Os números gerados não se repetem até que todas as possibilidades do intervalo sejam esgotadas.
- **Acessibilidade com Voz:** Integração com a API `ResponsiveVoice.js` para leitura automática dos textos da tela em Português do Brasil.
- **Feedback em Tempo Real:** Dicas visuais e sonoras indicando se o palpite deve ser maior ou menor.
- **Design Responsivo:** Interface moderna com background em gradiente, estilizada para se adaptar perfeitamente a monitores desktop e telas menores (mobile).

## 🛠️ Tecnologias Utilizadas
- **HTML5:** Estruturação semântica da página.
- **CSS3:** Estilização avançada com Flexbox, gradientes, efeitos de sombra e *Media Queries* para responsividade.
- **JavaScript (Vanilla):** Lógica de programação, manipulação do DOM (Document Object Model) e controle de estado do jogo.
- **ResponsiveVoice JS:** Biblioteca externa para *Text-to-Speech* (conversão de texto em fala).
